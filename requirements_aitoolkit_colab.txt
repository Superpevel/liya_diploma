# Зависимости для ai-toolkit (SDXL + FLUX LoRA) на Colab Python 3.12.
#
# Правила, выработанные кровью:
#   1. Не трогаем numpy/scipy/torch/torchvision — берём те, что в Colab.
#      Любая попытка их апгрейдить/даунгрейдить ломает ABI уже импортированных
#      C-расширений ("cannot load module more than once per process").
#   2. Версии пиннуем явно. Без пинов pip каждый релиз тянет случайные
#      апдейты, и в один прекрасный день albumentations 2.0 решит, что
#      ему нужен numpy<2 — и всё горит.
#   3. k-diffusion НЕ здесь — его ставит ячейка 1 отдельно с --no-deps,
#      потому что у него в зависимостях `clean-fid`, который под py3.12
#      не собирается (старый setup.py, тянет sklearn-сборку).
#   4. opencv-python-headless вместо opencv-python — у headless нет GTK/Qt
#      зависимостей, ставится за секунды, для тренировки этого хватает.
